# Computer Vision Image Processing

This repository contains image processing implementations and results using `Python`, `NumPy`, `OpenCV`, and `Matplotlib`.

## Main Files

- `Code.ipynb`: notebook containing the implementations and results

## Folders

- `images/`: input images used for the assignment
- `outputs/`: saved result images generated from the notebook

## How to Run

1. Open `Code.ipynb` in Jupyter Notebook or VS Code.
2. Run the cells in order.
3. Check the `outputs/` folder for saved figures and processed images.
